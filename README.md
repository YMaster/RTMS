# RTMS

## 简介
**RTMS** 使用 React + TypeScript + Mobx + Sass 搭建而成的一个项目

## 使用
```bash
npm run start
npm run build

# yarn
yarn start
yarn build

```

## 爬虫处理
见 robot.txt 配置

## 用到的库
emotion         css-in-js
axios           请求库
mobx            数据管理
normalize.css   css 初始化
